# _andamio — Andamio de construcción (descartable)

Todo lo que hay acá fue el **andamio para CONSTRUIR Sigil**: documentación de diseño por fases,
runbooks, playbooks de ALM, especificaciones originales y spikes de investigación. **No se mantiene
y probablemente se elimine.**

La documentación **viva** del proyecto es **[`docs/guias/`](../docs/guias/00-indice.md)**.

## Contenido

- `docs-diseno/` — documentación de construcción (fase-0, fase-1, fase-3, alm, runbooks, despliegues).
- `especificaciones/` — especificación original y guía del foro.
- `spikes/` — pruebas de concepto (PDFSharp/alpha, TSA RFC 3161, sandbox de plugins).

> Se conserva por valor histórico y de referencia mientras dure la transición. No dependas de esto
> para operar ni extender Sigil: para eso están las guías.
